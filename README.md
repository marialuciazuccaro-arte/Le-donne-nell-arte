# Le donne nell’arte — versione 3

Questa versione contiene:
- home;
- timeline cronologica;
- indice generale delle artiste con ricerca e filtro;
- pagine dei secoli;
- una pagina HTML per ciascuna artista;
- libreria locale delle immagini raccolte;
- file dati `data/artists.json` e `data/timelinejs_all.csv`.

## Come aggiornare i link Canva
Apri `data/artists.json` e inserisci l’URL pubblico Canva nel campo `canvaUrl` dell’artista desiderata.

## Come cambiare l’immagine di un’artista
Nel file `data/artists.json` modifica `localImage` (per immagini locali) oppure `media` (per immagini remote).

## Pubblicazione
Caricare l’intero contenuto della cartella nel repository GitHub Pages, sostituendo la versione precedente.
